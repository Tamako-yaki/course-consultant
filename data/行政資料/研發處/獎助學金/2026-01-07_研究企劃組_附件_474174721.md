# 國立臺北科技大學研究獎助生關係型態同意書

本表自106年8月1日起適用。

## 一、依據
依教育部「專科以上學校獎助生權益保障指導原則」及本校「學生兼任助理學習與勞動權益保障處理辦法」相關規定辦理。

## 二、著作權
本校教師於推動課程學習活動產生之相關研究成果著作權之歸屬，由研究獎助生與指導教授事先簽訂契約或依教育部相關規定辦理。

## 三、保險
研究獎助生從事研究學習活動期間，除原有學生團體保險外，計畫主持人應事先為其加保商業保險，以增加其保障範圍。保險期間自研究奬助生申請日起始，不得追溯承保。

## 四、學術倫理教育
自106年12月1日起，首次參與國科會計畫之研究獎助生，請檢附修習六小時之學術倫理教育課程訓練證明文件。相關課程可至「臺灣學術倫理教育資源中心」網站線上研習並下載學習時數證明。

## 五、簽名確認
本同意書經指導教師與學生同意認定後簽名，並送系所單位主管簽核後確定。

## 型態分流要件
| 型態 | 研究獎助生(學習) | 兼任研究助理(僱傭) |
| ---- | ------------------ | ------------------- |
| 處理原則 | 教育部「專科以上學校獎助生權益保障處理原則」 | 勞動部「專科以上學校兼任助理勞動權益保障指導原則」 |
| 定義 | 獲研究獎助之學生為發表論文、研究實習、課程或符合畢業條件，參與與自身研究相關之研究計畫或修習研究課程。 | 受學校僱用之學生兼任研究助理，並受學校或計畫主持人指揮監督，從事協助計畫工作。 |
| 權利義務 | 依本校學生及相關規定辦理 | 依勞動基準法等勞動法令及本校相關規定辦理 |
| 差勤記錄 | 有參與計畫之紀錄 | 應按時簽到退並備有出勤記錄 |
| 研究成果歸屬 | 學生享有著作權，指導教授不參與內容表達之撰寫。 | 依著作權法及專利法規定，研究成果之專利權歸屬本校。 |

## 注意事項
教師應有指導學生學習專業知識之行為，並應於到職前辦理勞(健)保事宜，工資、工時及延長工作時間應符勞動法令規定。
