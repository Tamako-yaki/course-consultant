# 學雜費減免作業須知

# 1. 學雜費減免

## 申請對象
- 原住民族學生
- 軍公教遺族
- 現役軍人子女
- 特殊境遇家庭之子女
- 身心障礙學生(人士子女)
- 低收入戶子女
- 中低收入戶子女

## 申請條件
- 以修業年限為限：
  - 五專部修業年限：1~5年級。
  - 大學部、碩士班修業年限：1~4年級。
  - 博士班修業年限：1~7年級。
- 同一教育單位不重複減免同年級重複就讀，第2次不享有減免資格。
- 政府機關同質補助不可重複請領。

## 申辦時間
- 115年01月01日起至115年01月30日截止。
- 研究所提前入學學生：至115年2月26日截止。
- 資格重新鑑定或因申復審查中而尚未確定者：至115年2月26日截止。

## 作業流程
1. 學生登入「校園入口網站-學務系統-學雜費減免及弱勢助學申請系統-學雜費減免系統」，填寫資料並確實送出。
2. 列印出「減免申請暨切結書」親筆簽名，併同「減免類別相對應之應備文件」書面資料於申請截止日前送至學務處。
3. 通過審查後，學雜費繳費單逕予減免。
4. 不通過可於申辦截止日前辦理資料補正作業。

## 應備文件
- 申請暨切結書
- 戶口名簿影本或戶籍謄本正本(3個月內有效)
- 撫卹令/卹亡給與令/年撫卹金證書影本（需有學生名字）
- 軍眷補給證影本
- 現役軍人身分證影本
- 有效期限內之特殊境遇家庭身分證明文件影本
- 有效期限之身心障礙手冊(影本)

## 減免額度
| 減免類別 | 學雜費減免額度 |
| --- | --- |
| 原住民族籍學生 | 工學院 22,800；商學院 20,400；文學院 20,200；平安保險免繳 |
| 軍公教遺族學生 | 工學院 19,330；商學院 18,008；文學院 17,888；全公費學生學(分)費&雜費減免100%；半公費學生學(分)費&雜費減免50% |
| 特殊境遇家庭之子女 | 學(分)費&雜費減免60% |
| 身心障礙學生 | 極重度/重度減免100%；中度減免70%；輕度減免40% |
| 低收入戶子女 | 學(分)費&雜費減免100%；平安保險免繳 |
| 中低收入戶子女 | 學(分)費&雜費減免60% |
