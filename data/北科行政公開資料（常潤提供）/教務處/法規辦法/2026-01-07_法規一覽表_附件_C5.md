# 國立臺北科技大學學生休退學學雜費收(退)費方式

# 國立臺北科技大學日間部大學部學生休退學學雜費收(退)費方式

## 一、新生
1. 申請休學須先完成註冊手續，包括繳驗學歷證件、繳納學雜費等費用、參加新生始業輔導。
2. 除學生平安保險費外，其餘各項費用依照「休學申請日之週次」予以退費，具體如下：

## 二、舊生
| 申請休學日期 | 已繳交者, 退還費用 | 未繳交者，應補繳納費用 |
|----------------|---------------------|--------------------------|
| 開學日前       | 全額【學費+雜費+網路使用費】 | 平安保險費               |
| ~開學後第1週(緩衝) | 1/3【學費+雜費+網路使用費】+平安保險費 | 2/3【學費+雜費+網路使用費】 |
| 第2週~第6週止 | 2/3【學費+雜費+網路使用費】+平安保險費 | 1/3【學費+雜費+網路使用費】+平安保險費 |
| 第7週~第12週止 | 1/3【學費+雜費+網路使用費】+平安保險費 | 2/3【學費+雜費+網路使用費】+平安保險費 |
| 第13週~第17週止 | 全額【學費+雜費+網路使用費】+平安保險費 | 不退費                   |
| 第18週(期末考週) | 禁止休學             |                          |

## 三、延修生
(註冊組約第3週提供選課資料給出納組製單，繳費期限約至第6週止)

| 選課否 | 申請休學日期 | 已繳交者, 退還費用 | 未繳交者，應補繳納費用 |
|--------|----------------|---------------------|--------------------------|
| 無     | 開學日前       | 全額【學分費+1/2 雜費+網路使用費】 | 平安保險費               |
|        | ~開學後第1週(緩衝) | 平安保險費               |                          |
|        | 第2週~第6週止 | 2/3【學分費+1/2 雜費+網路使用費】 | 1/3【學分費+1/2 雜費+網路使用費】+平安保險費 |
|        | 第7週~第12週止 | 1/3【學分費+1/2 雜費+網路使用費】 | 2/3【學分費+1/2 雜費+網路使用費】+平安保險費 |
|        | 第13週~第17週止 | 全額【學分費+1/2 雜費+網路使用費】 | 不退費                   |
