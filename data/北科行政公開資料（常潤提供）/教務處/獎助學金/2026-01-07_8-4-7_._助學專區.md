# 8-4-7 . 助學專區

助學專區包含以下內容：

1. [學雜費減免](https://oaa.ntut.edu.tw/p/412-1008-16046.php?Lang=zh-tw#top)
2. [弱勢助學金](https://oaa.ntut.edu.tw/p/412-1008-16046.php?Lang=zh-tw#1)
3. [就學貸款](https://oaa.ntut.edu.tw/p/412-1008-16046.php?Lang=zh-tw#2)
4. [其他助學措施](https://oaa.ntut.edu.tw/p/412-1008-16046.php?Lang=zh-tw#3)

## (一) 學雜費減免

符合下述資格之一者，且未重複申領減免或其他政府機關就學補助者：
- 軍公教遺族
- 原住民籍學生
- (中)低收入戶學生
- 身心障礙學生及身心障礙人士子女（家庭所得在220萬元以下）
- 現役軍人子女
- 特殊境遇家庭子女

### 作業期程
新生報到後可以進入校園入口網站起至114年2月5日前。

### 注意事項
- 上述學雜費減免，不得與其他就學減免補助重覆申請。
- 未完成相關申請者，請勿先繳費。
- 軍公教遺族就學優待與學雜費減免合併辦理，請於開學前完成申請。

### 應備證明文件
| 申請資格 | 證明文件 |
| --- | --- |
| 軍公教遺族 | 卹亡給與令、撫卹令、撫卹金證書；請同時攜帶減免學雜費申請暨切結書及各類資格之證明文件。 |
| 原住民學生 | 全戶戶籍謄本或族裔證明 |
| 中(低)收入戶 | 各鄉、鎮、市、區公所核發之(中)低收入戶證明文件影本或台北市之(中)低收入戶卡影本 |
| 身心障礙人士子女或學生 | 1.身心障礙手冊影本 2.全戶戶籍謄本影正本（需有父、母及同學本人）3.檢視前一年度家庭所得，所得須低於220萬。 |
| 現役軍人子女 | 1.軍人身份證、軍眷補給證 2.全戶戶籍謄本影正本 |
| 特殊境遇家庭子女 | 1.特殊境遇家庭證明文件影本 2.全戶戶籍謄本影正本 |

### 業務承辦窗口
學務處 課指組 莊小姐，分機 1286，E-mail：zoe@ntut.edu.tw

## (二) 弱勢助學金

同時具備下列4點條件同學，得申請本補助：
- 家庭應列計人口年所得在新臺幣70萬元以下。
- 家庭應計列人口之利息所得在新臺幣2萬元以下。
- 家庭應計列人口合計擁有不動產價值合計新臺幣650萬元以下。
- 前一學期學業成績平均高於60分。

### 作業期程
每學年上學期開學日起至第2週（週五）止。

### 注意事項
- 本補助於上學期申請，於下學期學雜費減免，且不得與其他就學減免補助重覆申請。

## (三) 就學貸款

已享有全部公費或學雜費全免同學不得申請就學貸款。僅享有部分減免學雜費同學得以學雜費減除減免之差額後申請就學貸款。

### 作業期程
114-2學期就學貸款受理時間：115年2月5日起至115年2月13日止；逾期未辦者不予受理。

### 注意事項
申辦就貸者，請勿繳交學雜費。

### 業務承辦窗口
學務處 生輔組 江怡葶小姐，分機1205

## (四) 其他助學措施
- 低收入戶學生免住宿費：請洽學務處李專員，分機1206。
- 本校學生急難慰問與濟助：詳情請至「獎助學金」-「急難救助」專區查閱，請洽學務處生輔組，分機1213。
- 校內外各項獎助學金及急難救助：詳情請至「獎助學金」專區查閱，請洽學務處莊小姐，分機1286。
- 高教深耕「琢玉計畫」獎助學金：詳情請至「琢玉計畫獎助學金」專區查閱，請洽學務處林小姐，分機1234。
