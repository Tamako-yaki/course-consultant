# 114學年度通識教育課程規範

## 課程架構
學生應於畢業前修滿通識課程共計 15 學分，分為：
- 學院指定向度課程：12 學分
- 學生自選向度課程：3 學分

### 各學院指定向度課程
| 向度名稱 | 學院 |
| -------- | ---- |
| 人文與藝術 | 機電學院、電資學院、工程學院、管理學院、設計學院、人社學院 |
| 社會與法治 | 機電學院、電資學院、工程學院、管理學院、設計學院、人社學院 |
| 自然與科學 | 機電學院、電資學院、工程學院、管理學院、設計學院 |
| 創新與創業 | 機電學院、電資學院、工程學院、設計學院 |

## 課程要求
1. 通識教育課程至少 28 學分，必修課程包含：
   - 大學入門 (1 學分)
   - 大學入門與工程倫理 (1 學分)
   - 語文閱讀與表達 (12 學分)
   - 博雅課程 (15 學分)
2. 語文閱讀與表達包含：
   - 國文 (2 學分)
   - 國語文實務應用 (2 學分)
   - 學院專業英文 (4 學分)
   - 跨領域專業英文 (4 學分)
3. 博雅課程修課規定：共 15 學分，分為學院指定向度課程 12 學分和學生自選向度課程 3 學分。
4. 學生每學期至多選修 2 門博雅課程，特殊情況下可加修第 3 門，需經通識教育中心主任審核同意。
